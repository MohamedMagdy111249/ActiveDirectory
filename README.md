# Active Directory User Management Script (PowerShell)

## Description
A PowerShell script designed for IT Help Desk and Junior Admin tasks. It automates common Active Directory lifecycle operations, including user creation, group assignments, account unlocks, and offboarding.

## Features
- Automates new user creation with secure password handling.
- Assigns users to specific Security Groups (RBAC).
- Unlocks accounts and handles password resets.
- Disables accounts during employee offboarding.

## Tools Used
- PowerShell / Active Directory Module (RSAT)
